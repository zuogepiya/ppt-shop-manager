# 如何找到 GitHub 仓库地址（图文指南）

## 步骤1：登录 GitHub
- 打开浏览器，访问：https://github.com/login
- 输入您的账号密码登录

## 步骤2：创建新仓库

### 2.1 找到创建按钮
登录后，看页面右上角：
```
+ [您的头像] ▼
    ↓
点击这个 "+" 符号
```

### 2.2 选择 "New repository"
点击 "+" 后会弹出菜单，点击 "New repository"

### 2.3 填写仓库信息
在创建页面，您会看到：

```
Repository name (仓库名称)
┌─────────────────────────────────┐
│ ppt-shop-manager                │  ← 在这里输入名称
└─────────────────────────────────┘

Description (可选)
┌─────────────────────────────────┐
│ PPT制作店铺员工管理系统          │
└─────────────────────────────────┘

Public ☑  Private ○  ← 选择 Public（公开）
```

### 2.4 点击创建按钮
页面底部有一个绿色按钮：
```
[ Create repository ]  ← 点击这个按钮
```

## 步骤3：找到仓库地址

创建成功后，您会看到一个新的页面，页面顶部或中间会有一个绿色或蓝色的框，里面显示：

```
Quick setup

HTTPS
┌─────────────────────────────────────────────┐
│ https://github.com/zuogepiya/仓库名.git    │ ← 这就是仓库地址！
└─────────────────────────────────────────────┘

[ Copy ]  ← 可以点击这个按钮复制

或者，页面会显示：
┌─────────────────────────────────────────────┐
│ ...or push an existing repository from the   │
│ command line                                │
│                                             │
│ git remote add origin https://github.com/  │
│   zuogepiya/仓库名.git                      │
│                                             │
│ git push -u origin main                     │
└─────────────────────────────────────────────┘
```

## 步骤4：复制仓库地址

**仓库地址格式：**
```
https://github.com/zuogepiya/您填写的仓库名.git
```

例如：
- 如果您填写的仓库名是 `ppt-shop-manager`
- 仓库地址就是：`https://github.com/zuogepiya/ppt-shop-manager.git`

## 重要提示

### 如果您看不到仓库地址页面
1. 检查是否真的创建了仓库
2. 可以访问：https://github.com/zuogepiya?tab=repositories 查看所有仓库
3. 点击任意一个仓库名称进入

### 如果仓库名称不是 ppt-shop-manager
没关系！只要是您创建的仓库就可以。
仓库地址就是：`https://github.com/zuogepiya/您的仓库名.git`

## 常见问题

**Q: 我已经创建了仓库，但找不到页面**
A: 访问 https://github.com/zuogepiya?tab=repositories 查看所有仓库

**Q: 我看到仓库了，但不知道地址是什么**
A: 仓库地址格式固定：`https://github.com/zuogepiya/仓库名.git`
  只要把仓库名替换成您创建的名称即可

**Q: 我可以创建多个仓库吗？**
A: 可以的！使用任意一个仓库都可以，只要告诉我仓库名
