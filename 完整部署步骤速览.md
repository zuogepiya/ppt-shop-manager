# 完整部署步骤速览 - 一页看懂

从零开始部署PPT店铺管理系统到公网。

**预计时间**：30-45分钟

---

## 📋 11个步骤（按顺序执行）

### 第1步：准备工作（5分钟）

**你需要准备**：
- ✅ 电脑（任何系统）
- ✅ 邮箱账号
- ✅ 浏览器

**不需要**：
- ❌ 编程知识
- ❌ 安装软件

---

### 第2步：创建Supabase数据库（5分钟）

**操作步骤**：

1. 访问 https://supabase.com
2. 注册账号并登录
3. 点击 "New Project"
4. 填写信息：
   - Name: `ppt-shop-manager`
   - Password: 设置强密码（⚠️ 记住！）
   - Region: 选择 Northeast Asia (Tokyo)
5. 点击 "Create new project"
6. 等待1-2分钟创建完成

**获取连接字符串**：
1. 点击左侧 Settings（⚙️）
2. 选择 Database
3. 找到 Connection string
4. 点击 URI 标签
5. 复制连接字符串并替换 `[YOUR-PASSWORD]`

**保存信息**：
- 数据库密码
- 完整连接字符串

---

### 第3步：创建GitHub账号（3分钟）

**操作步骤**：

1. 访问 https://github.com
2. 点击右上角 "Sign up"
3. 填写邮箱、密码、用户名
4. 验证邮箱
5. 记住你的GitHub用户名

---

### 第4步：推送代码到GitHub（2分钟）

**操作步骤**：

1. 访问 https://github.com/new
2. 创建新仓库：
   - Repository name: `ppt-shop-manager`
   - 选择 Private
3. 点击 "Create repository"
4. 记下你的GitHub用户名

**在当前环境执行2条命令**：

```bash
# 命令1：添加远程仓库（替换YOUR_USERNAME）
git remote add origin https://github.com/YOUR_USERNAME/ppt-shop-manager.git

# 命令2：推送代码
git push -u origin main
```

**验证**：
- 刷新GitHub仓库页面
- 能看到项目文件就成功了

---

### 第5步：注册Vercel账号（2分钟）

**操作步骤**：

1. 访问 https://vercel.com
2. 点击 "Sign Up"
3. 选择 "Continue with GitHub"
4. 授权Vercel访问GitHub
5. 设置Vercel用户名
6. 验证邮箱

---

### 第6步：部署到Vercel（5分钟）

**操作步骤**：

1. 登录Vercel
2. 点击 "Add New..." > "Project"
3. 导入 `ppt-shop-manager` 仓库
4. 点击 "Import"

**配置环境变量**：

| Key | Value |
|-----|-------|
| `DATABASE_URL` | 你的Supabase连接字符串 |
| `NODE_ENV` | `production` |

**部署**：
1. 点击 "Deploy" 按钮
2. 等待1-3分钟
3. 看到绿色对勾 ✅
4. 记录访问地址：`https://你的项目名.vercel.app`

---

### 第7步：初始化数据库（2分钟）

**操作步骤**：

1. 访问：`https://你的项目名.vercel.app/api/db-init`
2. 应该返回：`{"success": true, "message": "数据库表结构初始化成功"}`

3. 访问：`https://你的项目名.vercel.app/api/test-data`
4. 应该返回：`{"success": true, "message": "测试数据插入成功"}`

---

### 第8步：测试系统（2分钟）

**操作步骤**：

1. 访问：`https://你的项目名.vercel.app`
2. 使用测试账号登录：

| 角色 | 用户名 | 密码 |
|------|--------|------|
| 总台管理员 | `admin` | `admin123` |
| 管理者 | `manager` | `manager123` |
| 客服 | `cs` | `cs123` |
| 设计师 | `designer` | `designer123` |

3. 测试各个功能菜单

---

### 第9步：修改密码（⚠️ 必做）

**操作步骤**：

1. 使用 `admin` 账号登录
2. 进入"用户管理"
3. 为每个用户修改密码：
   - admin
   - manager
   - cs
   - designer

**密码建议**：
- 至少8位
- 包含大小写字母、数字

---

### 第10步：创建团队成员账号（5分钟）

**操作步骤**：

1. 使用 `admin` 账号登录
2. 进入"用户管理"
3. 点击"添加用户"
4. 填写用户信息：
   - 用户名
   - 密码
   - 角色（manager、cs、designer）
5. 保存

---

### 第11步：分享给团队（1分钟）

**分享信息**：

**访问地址**：
```
https://你的项目名.vercel.app
```

**账号信息**：
- 用户名：（你创建的用户名）
- 密码：（你设置的密码）

**使用说明**：
1. 在浏览器打开访问地址
2. 使用分配的账号和密码登录
3. 根据角色权限使用相应功能
