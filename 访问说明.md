## 如何访问系统

### 方式一：通过平台预览地址（推荐）

在Coze/字节跳动开发环境中，通常会提供一个**外部可访问的预览地址**。请查看你的开发平台界面：

1. 查看浏览器地址栏，是否有预览URL
2. 查看平台的"预览"、"部署"或"访问"按钮
3. 通常格式类似：`https://xxx.xxx.xxx:5000` 或其他代理地址

### 方式二：端口转发（如果使用SSH）

如果你是通过SSH连接到服务器的：

```bash
# 在本地终端执行端口转发
ssh -L 5000:localhost:5000 your-server
```

然后在浏览器访问：`http://localhost:5000`

### 方式三：直接访问（如果服务器有公网IP）

如果服务器有公网IP，直接访问：
```
http://your-server-ip:5000
```

## 如果仍然无法访问

### 检查清单

1. **查看开发平台界面**
   - 是否有"预览"或"访问"按钮
   - 是否显示了外部访问URL
   - 是否需要点击"刷新"或"重新部署"

2. **检查防火墙设置**
   ```bash
   # 检查防火墙是否允许5000端口
   sudo ufw status
   ```

3. **检查服务日志**
   ```bash
   # 查看Next.js开发服务器日志
   # 如果有错误信息，请截图反馈
   ```

4. **重启服务**
   ```bash
   # 停止当前服务
   pkill -f "next dev"

   # 重新启动
   cd /workspace/projects
   pnpm run dev --port 5000
   ```
